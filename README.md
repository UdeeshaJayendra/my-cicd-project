# My CI/CD Pipeline Project 

A production-grade CI/CD pipeline built with 
GitHub Actions and Docker.

## What this project does
- Automatically runs tests on every push
- Builds a Docker image automatically
- Pushes the image to Docker Hub automatically

## Tech Stack
- Node.js + Express
- Jest (automated testing)
- Docker
- GitHub Actions (CI/CD pipeline)
